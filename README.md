# Blackjack Game in Python

A simple **command-line Blackjack game** implemented in Python. Play against the computer and try to get as close to 21 as possible without going over!

## About

Blackjack (also known as 21) is a popular card game where the goal is to have a hand value as close to 21 as possible without exceeding it. The player competes against the computer (dealer).  

This Python version simulates the game using random card draws and basic Blackjack rules.

## Features

- Draw cards and choose to "hit" or "stand".  
- Automatic handling of **Ace** values (11 or 1).  
- Computer (dealer) plays automatically and hits until reaching 17 or more.  
- Announces the winner at the end of the game.
